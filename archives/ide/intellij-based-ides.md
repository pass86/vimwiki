IntelliJ-Based IDEs
===================
Let me love IDE again

# Recent Files
Command + E

# Search Everywhere
Double Shift

# Find in Path
Command + Shift + F

# Toggle Bookmark
F3

# Show Bookmarks
Command + F3

# Back
Command + [

# Forward
Command + ]

# Refactor Rename
Shift + F6

# Git Commit
Command + K

# Git Push
Command + Shift + K

# Maximize Tool Window
Command + Shift + '
